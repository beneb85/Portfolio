Vertical52

A redesign of a satellite image data platform for journalists into a clearer investigative workspace, improving workflows, navigation, and spatial data interaction.

Client Vertical52
Timeline 2026
Role UX / UI Designer
Impact International Reach

The Challenge

At its core, the challenge was to design for complex, data-heavy workflows while maintaining clarity and usability. Users needed to move seamlessly from data access to insight generation.

Key friction points included:

  • Overwhelming information density across market dashboards
  • No clear hierarchy between primary data and supporting context
  • Inconsistent patterns that slowed expert users and confused newcomers
  • A mobile experience that was an afterthought, not a first-class citizen

The platform needed to serve two very different users simultaneously: analysts who live in the data, and executives who need a 30-second read of the market.

The Solution

A redesigned information architecture that separates signal from noise — giving each user type an entry point that respects their context without hiding depth.

Key design decisions:

  • Progressive disclosure: summary cards surface key metrics first, detail layers reveal on demand
  • A unified component system that scales across six dashboard views without repetition
  • Contextual filters that persist across sessions, reducing repetitive setup time
  • Responsive-first layout that adapts data tables to mobile without losing functionality

The result is a platform that handles real estate complexity with the same directness a professional expects from their tools — fast, trustworthy, and built for daily use.

Information Architecture

New File Explorer

A centralized file explorer brings together investigations, satellite images, and workflows — eliminating context switching and improving control.

New Ways

Views

Users can switch between list, grid, and map views, allowing them to approach data from multiple perspectives and uncover insights more effectively.

New Features

The new map viewer enables users to open, layer, and compare multiple satellite images across different map backgrounds — turning static data into interactive analysis.
